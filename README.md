# Project Planner

## Project Setup
Follow these steps below to start-up json-server locally and serve app
### Steps
```bash
cd project-planner-vue
```

```bash
npm install
```

```bash
sudo npm i -g json-server
```

```bash
npm run json-sr
```

```bash
npm run serve
```
